Game Summary

Play as Estelle, a bunny witch, and collect different kinds of stardust while avoiding comets. Use your stardust to brew shielding, multiplier, magnet, and extra life potions, which give you powerups for your next adventure. Save up and unlock eight unique witch outfits so you can collect stardust in style!

Development Information

Tools: Unity, Photoshop, PaintTool SAI, Visual Studio
Duration: August 2019 - January 2020
Team Size: 1
Genre: Arcade
Platform: Mobile
Project Status: Completed - Get it on Google Play

Contributions

This was an independent game I created to learn more about mobile development. The work I did for the game includes:
- Developed prefabs for potions that allow for dynamic UI.
- Coded spawners for stardust, powerups, and comets using object pooling.
- Used PlayerPrefs to save player data such as score, currency, powerup, and closet information.
- Drew all characters, UI, and backgrounds, and implemented all art and UI through code.
- Designed and programmed four unique powerups, each with varying tiers, as well as eight unlockable outfits.
- Programmed all gameplay mechanics, including movement of characters, stardust, and obstacles.
