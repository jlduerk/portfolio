Game Summary

Miss Frost is a platformer where the player uses ice magic to freeze parts of the environment and get through each level. Using her freezing spell, Miss Frost can temporarily turn anything red, from plants to living creatures, into a frozen platform until it thaws. Find keys to unlock portals and move through the levels!

Development Information

Tools: GameMaker Studio 2, Photoshop, PaintTool SAI
Duration: December 2018 - January 2019
Team Size: 1
Genre: 2D Platformer
Platform: PC
Project Status: Demo Available

Contributions

This was an independent game I worked on to get a better understanding of the whole game development process. The work I did for the game includes:
- Concepted Miss Frost and created animated 2D sprites of her, as well as the other magical creatures.
- Created a 2D modular kit for the environment.
- Designed and developed each level.
- Programmed all gameplay mechanics, including movement, spells, and NPC movement.
- Designed and coded in UI and a load screen.

Whole levels were created on paper before they were recreated digitally using the kit I painted.

This is some earlier concept art for when I was designing Miss Frost. I gave her a relatively basic design so that her sprite animations would be simple and clean.
